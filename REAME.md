# Proyecto DAA - Unidad 2

Este repositorio contiene la entrega práctica correspondiente a la **Unidad 2** de la materia *Diseño y Análisis de Algoritmos*.

## Algoritmos Implementados

1. **Divide y Vencerás (Merge Sort):** Un algoritmo de ordenamiento recursivo con una complejidad temporal óptima de $O(n \log n)$ en el peor, mejor y promedio de los casos.
2. **Algoritmos Voraces (Mochila Fraccionaria):** Implementación de una solución Greedy que ordena los elementos según su relación valor/peso para maximizar la ganancia en una mochila con espacio limitado.

## Tecnologías Utilizadas
* HTML5
* JavaScript Vanila (ES6+)
* Tailwind CSS (Framework de Estilos vía CDN)
